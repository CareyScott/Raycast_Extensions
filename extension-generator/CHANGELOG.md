# Changelog

## [1.0.0] - 2025-01-05

### Added

- Initial release of Extension Generator
- Generate Raycast extensions using Claude Code CLI
- Analyze existing extensions for consistent patterns
- Prompt history with full tracking
- Edit and re-run previous prompts
- Progress tracking during generation
- 10-minute timeout with continuation prompt
- Detailed result view with file listing
- Quick access to open generated extensions in Finder/Terminal
- Support for custom Claude CLI path in preferences
